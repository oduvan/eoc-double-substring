Вы должны найти подстроку, которая повторяется больше одного раза в данной строке и при этом повторения друг на друга не накладываются. Например, в строке "abcab" самая длинная подстрока, которая повторяется более одного раза, это "ab", поэтому ответом должно быть число 2 (длина "ab").

Входные данные: Строка.

Выходные данные: Целое число.

Пример: 

{% if interpreter.slug == "js-node" %}
doubleSubstring('aaaa') == 2
doubleSubstring('abc') == 0
doubleSubstring('aghtfghkofgh') == 3 # fgh
{% else %}
double_substring('aaaa') == 2
double_substring('abc') == 0
double_substring('aghtfghkofgh') == 3 # fgh
{% endif %}
